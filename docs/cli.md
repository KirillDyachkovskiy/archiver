# Cli

### Flags

| Name      | Description             |               Type                |   Default   |
|:----------|:------------------------|:---------------------------------:|:-----------:|
| input     | Path to input file      |              string               |             |
| output    | Path to output file     |              string               |   `input`   |
| operation | Operation on files      |       `encode,`, `encode,`        |             |
| method    | File compression method |     `huffman`, `shannon-fano`     |             |
